# vue-biolerplate

> A Vue Web Project starter based on @vue/cli 3.

* Use vue-cli-plugin-vue-biolerplate, more flexible.
* Mock Data, use `lokijs`, `mockjs`, more creditable.
* authentication and authorization implemented.
* `BEM` base on `SASS`.
